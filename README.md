# TvT LMS - Hệ thống quản lý khóa học online

## Giới thiệu
TvT LMS là nền tảng học trực tuyến hiện đại, được xây dựng bằng React và Tailwind CSS.

## Tính năng
- ✅ Dashboard học viên
- ✅ Quản lý khóa học
- ✅ Chatbox tư vấn AI
- ✅ Tìm kiếm khóa học
- ✅ Responsive mọi thiết bị

## Cài đặt

```bash
# Cài dependencies
npm install

# Chạy ứng dụng
npm start
